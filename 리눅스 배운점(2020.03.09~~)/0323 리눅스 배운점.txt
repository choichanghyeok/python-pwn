1.프로모트 창에서 ex) choi@ubuntu라는것을 바꾸고 싶다면 PS1명령어를 사용하면된다.

ex ) PS1="\d$"   << 날짜를 나타냄

PS1="\t$" << 시간을 나타냄

여기서 []를 넣어주고싶다면?

PS1="[\t]$" 넣어주면됨.

(구글에 bash prompt escape 를 입력해서 \a나 등 여러명령어를 보고 자신만의 프로모트를 만들면됢)

재부팅하거나 종료하면 다시 원래대로 돌아옴


2. 프로필을 수정하는 명령어 관련

색깔을 바꾸는 명령어

구글에 텍스트 색상표를 검색해서 이용(검색내용 : LS_COLORS) 검색하면 여러 타입들에관한 내용도 나옴 ex)di (디렉토리)

33 = 오렌지색
ex) LS_COLORS = "di=0;33";

3. ls의 관해서 배움

ls에 명령어를 넣을수있음

1. type ls 		2. man ls 로 여러 명령어를 알수있음

3. ex) ls --color=always   	ls --color=auto		ls --color=never 
이것들이 가능

ll 이라는 명령어가 있음 ll(별칭) ls 라는 애한테 all등 여러개 기능이 있음

우리는 자주쓰는것들을 별칭으로 쓸수있음.

ex) alias ls= 'ls -l' 이러한식으로 별칭을 만듦
위에 예로 바꾸었을때 다시 원래대로 바꾸는법은
alias ls='ls --color=auto' 이다.

현재 이렇게 바꾸어도 프로필을 완전히 저장한것이 아니기때문에 로그아웃하고 다시들어오면 원래대로 돌아와있음.

이것을 완전히 저장하기위해서 영구적인 장소가 필요함. 다음 34강에서 배움.
