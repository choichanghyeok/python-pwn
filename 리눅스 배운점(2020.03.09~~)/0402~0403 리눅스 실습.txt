1. wget를 이용하여 jdk 를 설치하고 tar 을 이용해서 -zxvf 옵션을 이용해 압축을 풀어주었다.

2. 리졸빙을 이용해서 링크파일을 통해 java 의 버전과 javac를 이용한 컴파일러를 할수있었음

3. 텔넷 서비스를 이용해서 sudo ufw allow 23/tcp 를 통해서 23번 포트를 열고 윈도우에서 vm 리눅스의 ip주소를 입력해 접속했다

>> telnet 리눅스 ip주소

4. ssh 도 마찬가지로 sudo ufw allow 22/tcp 를 통해서 22번 포트를 열고 윈도우에서 주소를 입력하고 접속하였다.

>> 또 cmd 또는 putty 라는 프로그램을 통해 아이피를 입력해서 그 포트에 접속해 
	vm 리눅스에 접속할수있다는것을 실습했다.



## 동영상 내용대로 그대로 실습해야 했지만 vmware가 달라 포트 포워딩이 되지않아서
## ufw allow 명령어를 통해 해당 열고자하는 포트를 열어서 접속했다. 이과정은 내가 원하는 과정이 전혀 아니였다.
## 따라서 다른 리눅스 강의를 보고 다시 복습 해보자 한다.
