데이터베이스 서버

데이터 : 자료	, 테이블 : 데이터를 표형식으로 표현	, DB : 테이블의 집합		, DBMS : DB들을 관리하는 소프트웨어
레코드 : 테이블의 행	, 필드 또는 컬럼 : 테이블의 열		, 데이터 타입 : 각필드에 입력할 값의 형식(정수,문자 등)
필드명 : 각 필드의 이름	, 주키필드 : 레코드를 식별하기위해 유일한값을 가지고 비어있지 않은 필드
외래 키 필드 : 다른 테이블의 주키와 대응되는 필드	, RDBMS : 관계형 DBMS 	 
SQL : DB에서 정보를 얻거나, 생성 및 갱신하기 위해 정의된 표준 언어


DB와 관련된 SQL문

(MYSQL)
(개인적인 데이터베이스 수업에서 배운 자료가있음 보면서 참고)

DB 이름 조회

SHOW DATABASES;

사용할 DB지정

USE 데이터베이스 이름;

DB생성

CREATE DATABASE 데이터베이스 이름;

DB삭제

DROP DATABASE 데이터베이스 이름;


@@@@@@@@@@@@

테이블 이름조회

SHOW TABLES;

테이블구조 조회

EXPLAIN 테이블이름; OR DESC 테이블이름;

테이블 생성

CREATE TABLE 테이블 이름(필드이름1,필드타입1,필드이름2,필드타입2,...)

테이블 삭제

DROP TABLE 테이블이름;




@@@@@@



테이블 수정


ALTER TABLE 옵션

EX) ALTER TABLE customer MODIFY name CHAR(20);

레코드 삽입/삭제/수정 관련

삽입
INSERT INTO 테이블이름 VALUES (값1,값2,...);

삭제
DELETE FROM 테이블이름 WHERE 조건;

수정
UPDATE 테이블이름 SET 필드이름1 = 수정할값1,필드이름2 = 수정할값2..... WHERE 조건;

조회

SELECT 필드이름1,필드이름2....FROM 테이블이름 WHERE 조건;
